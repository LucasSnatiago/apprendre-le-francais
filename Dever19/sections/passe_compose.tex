\section{Conjuguez les Verbes Suivants au Passé Composé, dans Toutes Les Personnes}

    \item j'ai travaillé
    \item tu as travaillé
    \item il / elle / on a travaillé
    \item nous avons travaillé
    \item vous avez travaillé
    \item ils ont dansé
